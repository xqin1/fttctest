Using the Toll Calculator

Toll Rates

There are two ways to view toll rates, a list based view or a map view.

LISTS

  • For those preferring a list of rates, select Lists from the Main Menu. To assist in determining which Turnpike facility to view a menu of geographical areas is provided. After selection of the area, a list of available Turnpike facilities is provided.
  • Selection of a facility provides a comprehensive list of the exits and toll, if any.

MAPS

  • For those preferring a map view, select Maps from the Main Menu. To assist in determining which Turnpike facility to view a menu of geographical areas is provided. After selection of the area, a list of available Turnpike facilities is provided.
  • Selection of a facility highlights the facility and its exits/interchanges. Tapping the exit symbol provides a summary of the tolls at that particular location.

Toll Calculator

The toll calculator is used to calculate the total toll cost between two exits (interchanges) on Turnpike facilities. Users select their start point, then their end point, and the calculator will display a summary of the total toll charge.

There are two ways to calculate the toll cost of a trip, text based and map based.

TEXT SELECTION

  • For those preferring a text based toll trip calculation, select Calculator from the Main Menu, then Text Selection. To assist in determining which Turnpike facility on which to calculate a trip a menu of geographical areas is provided. After selection of the area, a list of available Turnpike facilities is provided.
  • Selection of a facility provides a list of available starting locations. Upon selection of a start location, the user will be prompted for an end exit. Finally, a summary of the trip is displayed, including the selected start location, end location, tolls incurred on the length of the trip, and a total toll charge.

MAP SELECTION

  • For those preferring a map based toll trip calculation, select Calculator from the Main Menu, then Map Selection. To assist in determining which Turnpike facility on which to calculate a trip a menu of geographical areas is provided. After selection of the area, a list of available Turnpike facilities is provided.
  • Selection of a facility displays a map of available starting locations. Upon selection of a start location, the user will be prompted for an end exit. Finally, a summary of the trip is displayed, including the selected start location, end location, tolls incurred on the length of the trip, and a total toll charge.
